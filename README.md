# Smart CCTV - Advanced Tactical HUD Monitoring

A futuristic, real-time security monitoring dashboard built with Python, CustomTkinter, and OpenCV. This application features an integrated tactical HUD overlay with interactive tracking modules.

---

## 🚀 Key Modules & Features

* **Engage Detection:** Implements real-time frame-differencing algorithms to track dynamic motions.
* **Scan Identity:** Initiates deep facial assessment cascades to verify local user directories.
* **Start Archive:** Captures direct feeds and handles automated video storage cycles.
* **Integrated Controls:** Built-in transparent interface zones for a sleek, modular design.
